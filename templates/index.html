Get started

    API Endpoint 
    https://api.kernelsnap.com/api/verify

This documentation offers programatic access to KernelSnap‘s API.
Please note that in order to be able to use the API an account should be created. API keys will be offered once the account is created.

To use this API, you need an API key. Please contact us at contact@kernelsnap.com to get your own API key.

Please note that any request will be temporarily stored in a queue. Once it is executed the output will be stored in a database and it will be permenantly deleted once the user retrieves the data

Execution time can vary between 4 minutes - 20 minutes

This operation will place the request in a queue until it gets executed. Once the request is received the user will receive an id for that specific request. Data should be transmitted as FormData.
https://api.kernelsnap.com/api/verify

QUERY PARAMETERS

Field Type Description
front File Image file corresponding to the front image of the id card
back File Image file corresponding to the back image of the id card
selfie File Image file corresponding to the selfie image of the person
public_key String Your public key
private_key String Your private key
country_code String Only TN is available for the moment
document_type String Only CIN is available for the moment
front_tolerance Integer Takes values between 1 and 3 (1 is recommended)
back_tolerance Integer Takes values between 1 and 3 (1 is recommended)
face_match_tolerance Integer Takes values between 1 and 3 (3 is recommended)
barcode_tolerance Integer Takes values between 1 and 3 (3 is recommended)
ocr_tolerance Integer Takes values between 1 and 3 (3 is recommended)
